## Setup (Windows)

```bash
# install enviroments
$ py -3 -m venv venv
```

## Running Locally (Windows)

```bash
# default https://localhost:5000
$ venv\Scripts\> activate
(venv) $ py main.py

# leaving venv
(venv) $ deactivate
```
